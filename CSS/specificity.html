Specificity

  • This example demonstrates on how specificity determines which rules are applied. The more specific the seletor the higher precedence it has.
  • There are 4 types of specificity. (order by most specific to least)
    1. The Style attribute (inline-style)
    2. The ID attribute
    3. The Class attribute
    4. The element itself (ie: div, li, span, ...)
  • Notes:
    1. The Style attribute has infinitely more specific value then any thing else. Meaning if an element has a style attribute set, those rules will overwrite all other rules it might have (from ids,classes and tags combined)
    2. The ID attribute has infinitely more specific value then classes and element tags. Meaning no amount of classes and/or element tags can outweight ID
    3. The Class attribute has infinitely more specific value then element tags. Meaning no amount of element tags can outeight a class
    4. The element tags have the lost specific value
  • In this example the element selector is set to color green, class to red, id to grey, and inline-style to blue.
