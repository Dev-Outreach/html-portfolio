Style Sheet vs Embedded vs Inline

  • In the specificty example we saw that inline styles will ALWAYS overwrite any rule it has in common with other selector (ID, class, or element), that is unless the other seletor has the important flag on the rule
  • In this example there are 3 intances of css styling for the class color-1.
    1. In an External style sheet sheet (demoStyleSheet.css) being referenced/added to this page in the head of the html
    2. In a style element before the html element
    3. In a style element after the html element
    Two intances of css style for the class color-2
    1. In an External style sheet sheet (demoStyleSheet.css) being referenced/added to this page in the head of the html
    2. In a style element before the html element
    Two intances of css style for the class color-3
    1. In a style element before the html element
    2. In a style element after the html element
    Two intances of css style for the class color-4
    1. In an External style sheet sheet (demoStyleSheet.css) being referenced/added to this page in the head of the html
    2. In a style element after the html element
  • Now for these 3 sources of css styling each source will set the color of the element
    1. All the classes in the External .css will set the color to blue
    2. All the classes in the style element before the html element will set the color to red
    3. All the classes in the style element after the html element will set the color to green
    This is to demonstrate which of instance of the classes (color-1, color-2, color-3, color-4) will be used to determine the color of the element
  • NOTE: the first 4 examples have an inline-style setting the color to black to show again that inline-style overwrites all

  • As shown in the above examples it doesnt matter where the classes (selectors) are define (embedded in a style element, or in an external style sheet). Class order still applies
  • Since the instances of the classes color-1, color-2, and color-3 that set the color to red are defined at the top before the other soucres, they will get overwritten.
  • Since the instances of the classes color-1, color-2, and color-4 that set the color to blue are defined in the external style sheet that is referenced/added to the pages inbetween the 2 style elements, its classes will overwrite the ones in the style element above it but get overwritten by the classes in the style element below it.
  • Since the instances of the classes color-1, color-3, and color-4 that set the color to green are defined at the bottom before the other soucres, they will overwrite all the classes above them.
